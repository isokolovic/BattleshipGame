# Battleship

Classic [Battleship](https://en.wikipedia.org/wiki/Battleship_(game)) board game - player vs. computer.

Built with [.NET 10](https://dotnet.microsoft.com/), [Avalonia UI 11](https://avaloniaui.net/) and [CommunityToolkit.Mvvm](https://learn.microsoft.com/en-us/dotnet/communitytoolkit/mvvm/).
